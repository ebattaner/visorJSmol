Una demo sobre estructura de biomoléculas (en Español)
A demo showing structures of biomolecules (in Spanish)
Visor JSmol – Demo de estructuras biomoleculares

Este repositorio contiene una demo interactiva para la visualización y exploración de estructuras biomoleculares en formato PDB utilizando JSmol, la versión HTML5/Javascript del proyecto Jmol.
Forma parte de un material docente para cursos de bioquímica, biología estructural y química orgánica.

Contenido del proyecto

Index.html — Página principal de la demo
/html/ — Páginas temáticas (isomería, configuraciones, comparaciones, etc.)
/pdb/ — Archivo de estructuras moleculares utilizadas en la demo
/gif/ — Imágenes de fórmulas químicas asociadas a las moléculas
/JSmol/ — Librería JSmol completa (JSmol.min.js y subcarpeta j2s/)

Abrir Index.html en el navegador
